HOW TO REPLACE TILE IMAGES

1) Open GitHub repository: luxeceram-site
2) Go to assets/tiles/
3) Upload your real tile photos with the exact same names:
   60x60-1.jpg, 60x60-2.jpg, 60x60-3.jpg
   50x50-1.jpg, 50x50-2.jpg, 50x50-3.jpg
   etc.
4) If you use JPG instead of SVG, edit the file paths in index.html and size pages:
   assets/tiles/60x60-1.svg  ->  assets/tiles/60x60-1.jpg

Best image size: 900x650 px or larger.
Use compressed JPG/WebP for faster loading.
